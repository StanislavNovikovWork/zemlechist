# Стандарты кода

## Организация компонентов

- **Каждый компонент хранится в отдельной папке** с именем компонента
- **Основной файл компонента называется `index.tsx`** и находится в папке компонента
- Это позволяет добавлять тесты, API, стили и другие файлы рядом с компонентом

### Пример структуры

```
components/
  map/
    Map/
      index.tsx
      Map.test.tsx
      Map.module.css
    MapSearch/
      index.tsx
    Marker/
      index.tsx
    types.ts
```

## Организация интерфейсов пропсов

- **Интерфейсы для пропсов компонентов** хранятся в файлах с компонентами, где они используются
- **Каждый интерфейс описывается с помощью JSDoc** на русском языке
- JSDoc должен содержать описание всех свойств интерфейса

### Пример

```typescript
/**
 * Пропсы компонента MapComponent
 * @property location - Начальное расположение карты с координатами центра и уровнем масштаба
 */
interface MapComponentProps {
  location?: { center: [number, number]; zoom: number };
}
```

## Общие типы

- Общие типы, которые используются в нескольких компонентах, хранятся в отдельном файле `types.ts` в соответствующей директории
