### UI Components Rule (Ant Design)

When working with UI components, layouts, or design systems, you MUST use Ant Design as the primary reference.

* Always consult the official Ant Design documentation: https://ant.design/components/overview/
* Prefer Ant Design components over custom implementations whenever possible
* Follow Ant Design patterns, props, and usage examples strictly
* Do not invent component APIs if an Ant Design equivalent exists
* Ensure consistency with Ant Design styling and structure

If a required component is not found in Ant Design:

* Clearly state that no matching component exists
* Then proceed with a custom implementation
